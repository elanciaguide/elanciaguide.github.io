# 일랜시아 가이드 — Project ER

**일랜시아(Elancia)**는 중세 판타지 세계관의 MMORPG입니다. 본 프로젝트(가칭 **Project ER**)는 넥슨 리플레이를 통해 일랜시아 IP 사용 계약을 맺고 새롭게 개발 중인 빌드이며, 현재 베타 테스트 단계로 콘텐츠와 시스템이 지속적으로 업데이트되고 있습니다.

> **베타 테스트 안내**: 현재 일랜시아는 베타 버전입니다. 게임 내 데이터는 정식 출시 시 초기화될 수 있으며, 가이드 내용은 패치에 따라 변경될 수 있습니다. 최신 변경사항은 [패치노트](/board/c/patchnote)에서 확인하세요.
